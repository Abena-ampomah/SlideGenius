# **App Name**: SlideGenius

## Core Features:

- Document Conversion: Convert documents and spreadsheets into structured presentation slides.
- Thematic Templates: Provide a library of presentation templates tailored for marketing, advertising, copywriting, social media, digital marketing, creative portfolio and AI consultant templates.
- AI Chatbot: Integrate an AI chatbot for user assistance and presentation tips, powered by Gemini.
- AI Image Generation: Allow users to generate custom images for their slides using Gemini-powered AI image generation.
- Automated Web Research tool: Enable AI-powered web research to gather relevant data and statistics for presentation topics. It's incorporated when LLM reasons it's appropriate
- FAQ AI: Offer an AI-driven FAQ section that answers user questions based on input data and previous queries.
- Leads Funnel: Implement a lead capture form to collect email addresses for sign-ups and marketing.

## Style Guidelines:

- Primary color: Vibrant magenta (#E00078) to convey boldness and innovation.
- Background color: Light pink (#F8BBD0) for a soft and friendly feel.
- Accent color: Electric blue (#7DF9FF) to draw attention to key elements.
- Body and headline font: 'Georama', a geometric sans-serif for a modern and fashionable feel.
- Use bold, geometric icons that complement the overall modern aesthetic.
- Implement a dynamic and engaging layout with clear calls to action and a user-friendly design.
- Incorporate subtle animations to guide users and add a touch of interactivity.
